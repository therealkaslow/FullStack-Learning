elementos flutuantes

podemos definir as divs como elementos flutuantes com a propiedade chamada float no CSS que consegue deslocar a div do lugar padrão para o lugar que foi destinado a ir

Outro elemento inline

temos outro elemento que está na mesma linha da div

Float

uma outra vantagem do float é que ele consegue automaticamente mandar toda a div para baixo caso a tela fique menor

propiedade clear

a propiedade clear serve para conseguir limpar todo o fluxo de elementos flutuantes para que o container pai consiga acompanhar o fluxo de modelo de caixa.
